## 9. 全局通用样式类（节选）

这些类定义在 `Index.html` 的 `<style>` 中，复用于多处：

| 类名 | 含义 |
|---|---|
| `.fill-背景颜色-20_45` | 主背景色 `rgba(29,36,44,1)` |
| `.fill-导航栏背景颜色-20_52` | 导航项背景 `rgba(34,41,50,1)` |
| `.fill-导航栏点亮时背景颜色-20_56` | 导航选中态 `rgba(236,105,153,1)`（粉色） |
| `.fill-导航栏字体颜色-20_48` | 导航文字色 |
| `.fill-设置左侧边栏背景颜色-20_61` | 设置页左侧栏背景 |
| `.fill-设置栏目点击时字体颜色-20_64-paragraph` | 设置栏目选中文字色 |
| `.fill-页面标题文本颜色-20_67-*` | 页面标题文字色（白） |
| `.text-页面内标题字体大小-20_74` | 标题字号 36px（思源黑体 Heavy） |
