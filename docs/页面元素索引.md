# 页面元素索引（id 字典）

> 配合 [`项目说明书.md`](./项目说明书.md) 使用。当前主入口为根目录 `Index.html` + `js/app.js`，下表以 **语义化 id / data 属性** 为主；文末保留旧版 Pixso 数字 id 索引供对照。
> 检索方式：在 `Index.html` 中搜索 `id="..."`，样式在 `css/styles.css` 中按类名检索。

## 1. 主 Tab 与容器（当前实现）

| 页面 | 容器 id | 说明 |
|---|---|---|
| 登录 | `#login-screen` | 全屏登录，成功后加 `.hidden` |
| 首页 | `#tab-首页` | 当前伙伴、屏幕权限、最近游戏 |

| 顶栏 Tab | `.nav-tab[data-tab]` | `首页` / `伙伴` / `游戏` / `设置` |

## 11. 设置页（`#tab-设置`，React Frame1762 / 46_273~46_377 参考）

### 布局与导航

| id / 选择器 | 含义 |
|---|---|
| `.settings-layout` | 左栏 + 右内容区 flex 容器 |
| `.settings-sidebar` | 左侧栏背景 `#191F25` |
| `#settings-nav` | 子栏目导航容器 |
| `.settings-nav-item[data-settings]` | 子栏目按钮；选中 `.active`（粉色 `#CA5C86`） |
| `.settings-content` | 右侧可滚动内容区 |
| `.settings-panel` | 子栏目面板；隐藏加 `.hidden`，id 为 `settings-panel-{子栏目名}` |

**子栏目名**（与 `data-settings` 一致）：`个人资料` / `快捷键设置` / `通用设置` / `画面识别权限` / `语音设置` / `用户反馈`

**切换方式**：`CompanionUI.setSettingsSubtab('通用设置')` 或点击 `.settings-nav-item`

### 个人资料

| id | 含义 |
|---|---|
| `#settings-profile-form` | 资料表单 |
| `#settings-avatar` | 头像 `<img>` |
| `#btn-change-avatar` | 「更改」头像（当前 Toast 占位） |
| `#settings-user-id` | 用户 ID（只读） |
| `#settings-nickname` | 昵称输入 |
| `#settings-gender-group` | 性别单选（男/女/保密） |
| `#settings-birthday` | 生日 `<input type="date">` |
| `#btn-save-profile` | 保存资料 |

### 通用设置

| id | 含义 |
|---|---|
| `#setting-auto-launch` | 开机自启动开关 |
| `#setting-dialogue-enabled` | 游戏内主动对话频率总开关 |
| `#setting-dialogue-frequency` | 频率滑块（1=较低, 2=中等, 3=较高） |
| `#setting-frequency-text` | 当前频率文案 |
| `#settings-frequency-wrap` | 频率区域；开关开启时 `.is-active` |
| `#setting-auto-interrupt` | 自动打断开关 |

### 画面识别权限

| id | 含义 |
|---|---|
| `#setting-screen-enabled` | 启用画面识别总开关 |
| `#settings-permission-list` | 与首页 `#permission-list` 同步的权限列表 |
| `#btn-add-permission-settings` | 添加权限（打开 `#modal-add-permission`） |

### 语音设置

| id | 含义 |
|---|---|
| `#setting-voice-language` | 伙伴输出语言 `<select>` |
| `#setting-voice-volume` | 音量滑块 0–100 |
| `#setting-volume-text` | 音量数值显示 |

### 用户反馈

| id | 含义 |
|---|---|
| `#settings-feedback-text` | 反馈正文（最多 800 字） |
| `#settings-feedback-count` | 字数统计 `0/800` |
| `#btn-feedback-image` | 添加图片（当前 Toast 占位） |
| `#btn-submit-feedback` | 提交反馈 |

### 状态与持久化（`app.js`）

| 状态对象 | localStorage 键 | 说明 |
|---|---|---|
| `userProfile` | `companion-ui-settings`（JSON 内） | `userId` / `nickname` / `gender` / `birthday` / `avatar` |
| `generalSettings` | 同上 | `autoLaunch` / `dialogueEnabled` / `dialogueFrequency` / `autoInterrupt` / `screenRecognitionEnabled` |
| `voiceSettings` | 同上 | `language` / `volume` |
| `activeSettingsSubtab` | 同上 | 上次打开的子栏目 |

**事件**：`settings:subtab-change` / `settings:profile-save` / `settings:general-change` / `settings:voice-change` / `settings:feedback-submit`
